# PlaylistCheck 🎵

একটি স্বয়ংক্রিয় প্লেলিস্ট জেনারেটর এবং লিঙ্ক চেকার। এই রিপোজিটরিটি GitHub Actions-এর মাধ্যমে নির্দিষ্ট সময় পর পর
`channels.txt` ফাইল থেকে লাইভ চ্যানেলগুলোর স্ট্যাটাস চেক করে এবং সচল লিঙ্কগুলো নিয়ে একটি ফ্রেশ `combined_playlist.m3u` ফাইল তৈরি করে।

## বৈশিষ্ট্য (Features)
* 🔄 **অটো-আপডেট:** প্রতি ৪ ঘণ্টা পর পর প্লেলিস্ট স্বয়ংক্রিয়ভাবে আপডেট হয়।
* 🔍 **লিঙ্ক ভ্যালিডেশন:** সচল এবং ডেড লিঙ্কগুলো স্বয়ংক্রিয়ভাবে ফিল্টার করা হয়।
* 📱 **সর্বজনীন সামঞ্জস্য:** তৈরি হওয়া `.m3u` প্লেলিস্টটি যেকোনো IPTV প্লেয়ারে (যেমন- VLC, OTT Navigator, Tivimate) সরাসরি ব্যবহার করা যায়।

---

## যেভাবে ব্যবহার করবেন (How to Use)

আপনার প্রিয় IPTV প্লেয়ারে সরাসরি নিচের যেকোনো একটি পদ্ধতি ব্যবহার করে প্লেলিস্টটি যুক্ত করুন:

### ১. লাইভ নেটওয়ার্ক URL (অনুরোধ করা হচ্ছে)
সবচেয়ে ভালো অভিজ্ঞতার জন্য সরাসরি এই অনলাইন লিঙ্কটি আপনার প্লেয়ারে কপি-পেস্ট করুন। এর ফলে প্লেলিস্ট আপডেট হলে আপনার প্লেয়ারেও চ্যানেলগুলো স্বয়ংক্রিয়ভাবে আপডেট হয়ে যাবে।
